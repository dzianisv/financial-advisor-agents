## research-market workflow bugs fixed (2026-06-20)

### Bug 1: ARGS key mismatch (PR #46, merged)
- `ARGS.question` was used in workflow but callers passed `ARGS.query` — every equities run silently defaulted to hardcoded BTC query
- Fix: normalize both keys + remove hardcoded crypto default
- Lesson: always validate ARGS field names match between workflow definition and callers; hardcoded fallbacks mask missing input

### Bug 2: Monolithic phase timeout (PR #47)
- Old architecture ran all 13 tickers in one monolithic gather/consolidate phase → stalled at 50+ min
- Fix: per-asset pipeline — each ticker gets dedicated gather/consolidate/panel agents spawned in parallel via pipeline(ASSETS)
- Lesson: fan out per-asset in parallel from the start; never serialize ticker loops in a single agent
